    다시 시도
    사용자가 참여 중인 프로젝트를 한눈에 보고 진입하거나 새로 생성한다.
  • GitHub collaborator인 저장소와 매칭되는 프로젝트만 노출 (public이어도 collaborator 아니면 미표시)
  • 즐겨찾기 상태는 사용자별이며, 다른 사용자에게 영향이 없다
  • 정렬: 즐겨찾기 우선 → 생성일 내림차순
  • 프로젝트 0개일 때만 예제 안내가 노출. fork 완료 후에는 일반 목록으로 전환
  • 헤더 프로필 아바타 클릭 시 [프로필 설정 시트](../../common/sheet_profile.md) 열림
  • 프로젝트 카드 클릭 시 `/projects/[id]`로 이동, 첫 번째 화면으로 자동 리다이렉트
