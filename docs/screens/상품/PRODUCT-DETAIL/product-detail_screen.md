# 상품 상세

## 개요

개별 상품의 상세 정보를 확인하고 구매할 수 있는 화면. 상품 이미지, 가격, 옵션 선택, 리뷰를 한 화면에서 제공한다.

## 기능 정의

| 기능 ID | 기능명 | 설명 | 우선순위 |
|---------|--------|------|---------|
| GALLERY | 이미지 갤러리 | 메인 이미지 + 썸네일 목록. 스와이프/클릭으로 전환 | P0 |
| INFO | 상품 정보 | 상품명, 가격, 할인율, 배송 정보 | P0 |
| OPTION | 옵션 선택 | 색상, 사이즈 등. 재고 없는 옵션 비활성화 | P0 |
| REVIEW | 리뷰 | 평점 분포, 리뷰 목록, 사진 리뷰 필터 | P1 |
| RELATED | 연관 상품 | 같은 카테고리 인기 상품 4개 | P2 |

## 화면 구성

### 상단: 이미지 갤러리

- 메인 이미지 영역 (PC: 좌측 50%, MO: 전체 너비)
- 썸네일 리스트 (PC: 메인 이미지 하단, MO: 가로 스크롤)
- 이미지 줌: PC에서 마우스 호버 시 확대

### 중단: 상품 정보 + 옵션

```
상품명: 프리미엄 무선 이어폰 Pro Max
브랜드: SoundTech
─────────────────────────
  89,000원  →  69,000원 (22% 할인)
─────────────────────────
색상: ● 블랙  ○ 화이트  ○ 네이비
사이즈: [선택하세요 ▾]
─────────────────────────
수량: [- 1 +]
─────────────────────────
[장바구니 담기]  [바로 구매]
```

### 하단: 리뷰

- 평점 요약: 평균 별점 + 별점 분포 바 차트
- 리뷰 필터: 전체 / 사진 리뷰 / 별점별
- 리뷰 카드: 작성자, 별점, 날짜, 본문, 사진

## 비즈니스 규칙

> **할인가 표시 규칙**: 할인율이 5% 이상일 때만 할인가와 원가를 동시 표시한다. 5% 미만이면 할인가만 표시.

- 재고 0인 옵션은 **취소선 + 회색 처리**
- 모든 옵션 재고 0이면 구매 버튼 비활성화 + "일시 품절" 표시
- 장바구니 담기 성공 시 토스트: "장바구니에 담았습니다" + [바로가기] 링크
