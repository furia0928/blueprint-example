# 로그인

## 개요

서비스 진입점. 이메일/비밀번호 로그인과 소셜 로그인을 지원한다.

## 기능 정의

| 기능 ID | 기능명 | 설명 | 우선순위 |
|---------|--------|------|---------|
| EMAIL_LOGIN | 이메일 로그인 | 이메일 + 비밀번호 조합 | P0 |
| SOCIAL_LOGIN | 소셜 로그인 | 카카오, 네이버, Google | P0 |
| FIND_PW | 비밀번호 찾기 | 이메일 인증 후 재설정 | P1 |

## 화면 구성

- 로고 (중앙 정렬)
- 이메일 입력 필드
- 비밀번호 입력 필드 (show/hide 토글)
- [로그인] 버튼 (primary)
- "비밀번호를 잊으셨나요?" 링크
- 구분선 "또는"
- 소셜 로그인 버튼 3종

## 유효성 검증

| 필드 | 규칙 | 에러 메시지 |
|------|------|------------|
| 이메일 | 이메일 형식 필수 | "올바른 이메일 주소를 입력해주세요" |
| 비밀번호 | 8자 이상 | "비밀번호는 8자 이상이어야 합니다" |
| 로그인 실패 | 5회 초과 시 30분 잠금 | "로그인 시도 횟수를 초과했습니다. 30분 후 다시 시도해주세요" |

## 접근성

- 폼 필드에 적절한 `label` 연결
- Enter 키로 폼 제출
- 에러 메시지에 `role="alert"` 적용
- 소셜 로그인 버튼에 서비스명 포함 (`aria-label="카카오로 로그인"`)
